## 🚀 The Project I'm most proud of:

### FleetForward – Fleet Management System

FleetForward is a **fleet management platform** designed to help companies efficiently manage their vehicles, drivers, and logistics operations.

**Features**
- Manage and track **fleet vehicles**
- Monitor **driver assignments and vehicle usage**
- **Client-server communication** for real-time data updates
- **Blazor user interface**
- **.NET used for the business layer**
- **Java used for the data layer**
- **PostgreSQL database integration**
- Structured using the **MVC architecture**

The goal of FleetForward is to provide a **reliable and scalable system for managing transportation resources efficiently.**
